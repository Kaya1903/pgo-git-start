# pgo-git-start

Author: Doruk Uslu

This repository was created for the PGO Git exercises.
The goal is to practice Git, GitHub, branches and merging.
## Repository structure

- README.md
- src/Main.java

## Commands used

- git init
- git status
- git add
- git commit
- git checkout -b
- git merge
- git push

## Exercise stages

1. Created the local repository
2. Added README.md and Main.java
3. Made the initial commit
4. Connected the repository to GitHub
5. Created a feature branch
6. Updated the README file
